# Mika
Mika Game Engine\
Now only support Windows platform.

## Graphic API:
1. OpenGL (development)
2. Vulkan (Not Yet)

## How to build on Windows:
1. Install Visual Studio 2022.
2. Get this repository.
3. Execute gen_project.bat.
4. Open Mika solution.
